上传
git init
git remote add origin 地址  从本地上传到远程库中

下载
git clone 地址  从远程库中克隆到本地

git add . :添加全部新文件

git commit -m "add" 提交到本地的版本控制库里

git push 提交到服务端去

git diff 查看差异性

git status: 命令可以让我们时刻掌握仓库当前的状态，上面的命令告诉我们，readme.txt被修改过了，但还没有准备提交的修改。


git pull 更新
